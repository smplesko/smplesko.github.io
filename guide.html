Getting Started

Welcome to the Dird Plesk Memorial Open Invitational of Champions tracker! This site helps manage scores, teams, and standings across multiple events.

For Players

  1. Enter the site password when prompted (provided by your event organizer)
  2. Select your name from the login screen on the home page
  3. View your profile, check standings, and make predictions
  4. Follow along as scores are updated in real-time

For Admins (First-Time Setup)

  1. Log in as the admin user (requires admin password)
  2. Go to the Admin page from the footer
  3. Set up player names (up to 12 players)
  4. Configure events and scoring as needed
  5. Add trivia questions (manually or via CSV upload)
  6. Create predictions for bonus points

Player Login & Profiles

On the home page, you'll see a grid of player buttons. Tap your name to log in.

Profile Features

  • Point Breakdown: See your points from each event
  • Current Rank: Your position in the overall standings
  • Team Assignments: View which teams you're on for each event
  • Event History: Detailed breakdown of your performance

To log out, click the "Logout" button in the header bar.

Events Overview

The competition features five ways to earn points:

Golf

Team-based golf scoring with flexible team configurations.

How It Works

  • Players are assigned to teams (2, 3, 4, 6, or 12 teams)
  • Teams compete, and finishing positions determine points
  • All team members receive the same points based on team placement

Bonus Points

  • Best Dressed: Awarded to the snazziest golfer
  • Longest Drive: Furthest drive on designated hole
  • Closest to Pin: Nearest to the pin on par 3
  • Most Honest: Best sportsmanship award

Beer Olympics

Five separate games with individual scoring per game.

How It Works

  • Each game can have different team configurations
  • Points are awarded based on team finishing position in each game
  • Individual players earn points based on their team's performance
  • All 5 game scores combine for total Beer Olympics points

Scoring

Points are distributed based on placement (1st, 2nd, 3rd, etc.). The exact point values can be configured by the admin.

Go-Karting

Individual racing competition.

How It Works

  • Each player competes individually
  • Finishing positions determine points
  • Admin enters final positions after racing

Point Structure

Points decrease by position (e.g., 1st: 12 pts, 2nd: 10 pts, etc.). Configurable by admin.

Trivia

Live interactive trivia with real-time scoring.

How It Works

  • Admin starts the trivia game from the admin panel
  • Players join the waiting room
  • Questions appear one at a time on all screens
  • Players select answers before time runs out
  • Correct answers earn points

Question Types

  • Multiple Choice: Select from predefined options (auto-graded instantly)
  • Open-Ended: Type your answer (admin reviews and approves)

Features

  • Categories: Questions can display a category badge (e.g., "History", "Sports")
  • Auto-Grading: Multiple choice questions are automatically scored
  • Real-time Sync: All players see questions simultaneously
  • Live Scoring: Points update instantly

Predictions

Make picks on various outcomes for bonus points.

How It Works

  • Predictions are created by the admin with multiple choice options
  • Players submit their picks before the prediction is locked
  • Once locked, no more answers can be submitted
  • Admin selects the correct answer when the outcome is known
  • Players who predicted correctly earn bonus points

Important Notes

  • Double Confirmation: You'll be asked to confirm your pick to prevent accidental submissions
  • No Changes: Once submitted, your answer cannot be changed
  • Privacy: Other players' picks are hidden until you submit yours
  • Banner Alerts: A notification will appear when you have unanswered predictions

Leaderboard

Track standings across all events in real-time.

Overall Standings

Shows total points from all events combined, with individual event columns.

Event Breakdowns

  • Golf Results: Team standings and player points
  • Beer Olympics: Individual totals across all 5 games
  • Go-Karting: Race finishing order and points
  • Trivia: Individual scores from trivia questions
  • Predictions: Bonus points from correct picks

Final Podium

When the competition is closed by the admin, a podium display shows the top 3 finishers.

Admin Guide

This section covers all admin functionality. You must be logged in as the admin user to access the Admin page.

Managing Players

The admin can configure up to 12 player slots.

Setting Up Players

  1. Navigate to the Admin page
  2. Find the "Manage Players" section
  3. Edit player names in the input fields
  4. Click "Save Players" to update

Note: Player names are used across all events and the login screen.

Event Management

Each event has its own management section.

Golf

  1. Click "Golf" in Event Management
  2. Select number of teams (2, 3, 4, 6, or 12)
  3. Assign players to teams by tapping their names
  4. Save teams, then enter finishing positions
  5. Add bonus point winners (Best Dressed, Longest Drive, etc.)

Beer Olympics

  1. Click "Beer Olympics" in Event Management
  2. Select which game to manage (1-5)
  3. Set number of teams for that game
  4. Assign players to teams
  5. Save teams, then enter team placements
  6. Repeat for each game

Go-Karting

  1. Navigate to the Go-Karting page
  2. Enter finishing positions for each player
  3. Save results

Event Locks

Lock events to preserve results and prevent accidental changes after completion.

Available Locks

  • Golf: Locks team assignments and hole scores
  • Beer Olympics: Locks all 5 game results
  • Go-Karts: Locks race finishing positions
  • Trivia: Locks trivia game and responses
  • Predictions: Locks prediction responses and finalization

How to Lock/Unlock

  1. Go to Admin page
  2. Find "Event Locks" in Site Settings
  3. Toggle the switch for each event
  4. Locked events show a "(Locked)" indicator

Tip: Lock events after completing them to prevent accidental changes. You can always unlock if needed.

Trivia Setup & CSV Upload

Trivia questions can be added manually or uploaded in bulk via CSV. Supports both multiple choice (auto-graded) and open-ended questions.

Manual Entry

  1. In the Admin panel, find Trivia Management
  2. Enter the question text
  3. Set point value (1-10 points)
  4. Optionally add a category (displayed during trivia)
  5. Click "+ Add Multiple Choice Options" for auto-graded questions
  6. If multiple choice: enter options and select correct answer
  7. Save the question

Question Types

  • Open-Ended: Players type answers. Admin manually approves/denies each response.
  • Multiple Choice: Players select from options. Automatically graded - correct answers are instantly approved.

CSV Upload (Bulk Import)

For bulk import, prepare a CSV file. The format determines the question type:

Multiple Choice Format (Auto-Graded)

question,option1,option2,option3,option4,correct_answer,category
  • question: The trivia question text (required)
  • option1-4: Answer choices (at least 2 required, up to 4)
  • correct_answer: Number 1-4 indicating which option is correct (required)
  • category: Question category displayed as a badge (optional)

Freeform Format (Admin Approves)

question,category
  • question: The trivia question text (required)
  • category: Question category (optional)

If no options or no correct_answer is provided, the question becomes freeform.

Example CSV Content

"What is the capital of France?","London","Paris","Berlin","Madrid",2,"Geography"
"Who wrote Romeo and Juliet?","Dickens","Shakespeare","Hemingway","Austen",2,"Literature"
"Name three US presidents","History"
"What's your favorite vacation spot?","Fun"

Tips

  • Wrap text containing commas in double quotes
  • Header row is optional (auto-detected and skipped)
  • Maximum 16 questions will be imported
  • All imported questions default to 1 point each
  • You can mix multiple choice and freeform in the same CSV

Running the Trivia Game

  1. Ensure questions are added
  2. Players join the waiting room from the Trivia page
  3. Click "Start Trivia - Show Q1" when players are ready
  4. Players answer, then click "Reveal Responses"
  5. For multiple choice: correct answers are pre-approved
  6. For open-ended: approve/deny each response manually
  7. Advance to next question, repeat until complete
  8. Click "Finish Trivia" to show final results

Predictions Management

Create and manage prediction questions for bonus points.

Creating a Prediction

  1. In Admin, find the Predictions section
  2. Enter the prediction question
  3. Set the point value for correct answers
  4. Add answer options
  5. Save the prediction

Managing Predictions

  • Lock: Prevents new answers from being submitted
  • Finalize: Select the correct answer and award points
  • Delete: Remove a prediction entirely

Tip: Lock predictions before the event occurs, then finalize once you know the outcome.

Closing the Competition

When all events are complete, you can officially close the competition to reveal final standings.

How to Close

  1. Ensure all events are scored and locked
  2. In Admin, find "Close Competition"
  3. Confirm your decision
  4. The final podium will display on the Leaderboard page

Note: This is separate from ending individual events. Closing the competition reveals the overall winners with a podium celebration.

Data Management

Tools for backing up and resetting competition data.

Export Data

Downloads a JSON file containing all competition data including:

  • Player names and scores
  • All event results
  • Trivia questions and answers
  • Predictions and responses

Reset All Data

Warning: This permanently erases all competition data. Use only when starting fresh.

  1. Click "Reset All Data"
  2. Confirm by typing the required phrase
  3. All scores, teams, and results will be cleared
