## How to Use This File

This tracks design choices for the Dird Plesk Memorial project across three categories:

1. **Open Questions** — Choices that need Stephen's input before work continues
2. **Decided** — Resolved questions with the ruling and reasoning preserved
3. **Spec/Doc Update Candidates** — Things discovered during development that may need CODEBASE_REFERENCE.md or CLAUDE.md updated

**Rules:**
- Decision IDs use the format **CD-1, CD-2, CD-3...** (Code Decision) and are assigned sequentially — never reused or reassigned.
- Before assigning a new CD number, always check the current highest number in this file to avoid collisions.
- The Reasoning field is the most valuable part long-term. Be thorough.
